# Axon — AI Consulting Landing Page

> A production-grade landing page for an AI automation consulting business — dark editorial aesthetic, animated neural network hero, and conversion-focused layout. No frameworks. Single file.


## What This Is

## Design
- **Hero:** Animated neural network canvas with particle connections (vanilla Canvas API)
- **Sections:** Navigation · Hero · Client showcase · Services grid · Process timeline · Case studies · Testimonials · Team · Footer
- **Typography:** Cormorant Garamond + DM Mono + Outfit
- **Responsive:** Mobile-first across all screen sizes

## Tech Stack
- **Canvas API** — animated neural network background
- **IntersectionObserver** — scroll-reveal animations
- **Node.js** — local dev server (`serve.mjs`)

## Run It
git clone https://github.com/theokalogr-bit/Axon-ai-consulting-website
cd Axon-ai-consulting-website
node serve.mjs
# Open http://localhost:3000
```

## Use Cases
- **Agencies** — show clients what a modern, premium AI services page looks like before committing to a full build
- **Freelancers** — deploy as-is or fork and customize; the single-file architecture makes it trivial to modify without a build pipeline
